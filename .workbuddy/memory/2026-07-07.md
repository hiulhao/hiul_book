# 2026-07-07 工作日志

## 个人博客项目搭建

用户决定搭建个人博客网站，经过方案对比后选择了 SSG（静态站点生成器）方案。

### 技术选型
- 框架：Astro v4.16
- 样式：Tailwind CSS + @tailwindcss/typography
- 内容：Markdown + Astro Content Collections
- 部署目标：Vercel
- 评论：Giscus（待配置）
- 搜索：Pagefind（构建时生成索引）
- 收藏：localStorage 本地存储

### 已完成功能
- 项目初始化和配置（astro.config.mjs, tailwind.config.mjs, tsconfig.json）
- 全局样式 + 暗黑模式（CSS 变量 + class 切换）
- 布局组件：Header（导航 + 主题切换）、Footer、BaseLayout
- 首页：文章列表 + 搜索框 + 置顶文章标记
- 文章详情页：Markdown 渲染 + 上下篇导航 + 收藏按钮 + Giscus 评论
- 标签系统：标签云 + 标签筛选页
- 搜索功能：Pagefind 集成（需构建后生效）
- 收藏功能：localStorage 管理收藏列表
- RSS 订阅：/rss.xml
- 关于页、404 页
- 3 篇示例文章（含 1 篇置顶）

### 项目结构
```
src/
├── components/   Header, Footer, PostCard, Search, Comments, FavoritesScript, FormattedDate
├── content/
│   ├── config.ts   Content collection schema
│   └── posts/      3 篇示例 Markdown 文章
├── layouts/        BaseLayout
├── pages/
│   ├── index.astro         首页
│   ├── about.astro         关于
│   ├── favorites.astro     收藏页
│   ├── 404.astro
│   ├── rss.xml.js          RSS feed
│   ├── posts/[...slug].astro   文章详情
│   └── tags/
│       ├── index.astro     标签云
│       └── [tag].astro     标签筛选
└── styles/global.css
```
