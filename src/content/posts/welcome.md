## 你好

欢迎来到 **Hiul·Book**，我的个人技术博客。

这里会记录我在软件开发过程中的思考、学习和实践。不追求高频更新，但希望每篇文章都有价值。

## 为什么写博客

写博客是最好的学习方式。当你能把一个技术点讲清楚，说明你真正理解了它。费曼说的。

另外，博客也是一个长期积累的个人知识库。搜索引擎会告诉你"怎么做"，但博客会告诉你"为什么这么做"。

## 这个博客是怎么搭的

- **框架**：Astro，静态站点生成器
- **样式**：Tailwind CSS
- **内容**：Markdown 文件，Git 管理
- **部署**：Vercel，push 即上线
- **评论**：Giscus，基于 GitHub Discussions

整个站点没有任何数据库，没有服务器，加载极快。你看到的这个页面是一个纯静态 HTML 文件。

## 接下来会写什么

- 前端工程实践（Astro / React / 性能优化）
- 后端架构思考（Node.js / 数据库 / API 设计）
- 开发工具和效率技巧
- 偶尔的技术随笔
